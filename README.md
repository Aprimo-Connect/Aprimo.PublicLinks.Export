# Aprimo.PublicLinks.Export

### Configuration

##### app.config
- Set the values within <appsettings>
   - AprimoUsername
   - AprimoClientID
   - AprimoClientSecret
   - AprimoTenant
   
##### Program.cs
- destinationFolder - The folder destination the export should be saved to
- nameofCSV - the name of the exported CSV to be saved to destinationFolder
